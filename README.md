# Vortex Hub

Vortex Hub es una red social orientada a desarrolladores de Software, programadores, y entusiastas de la tecnologia.

Vortex Hub está diseñada para ser útil, segura, y única. Nuestra principal prioridad es la seguridad de los usuarios, es por eso que hemos decidido tomar cartas en el asunto e implementar un sistema de Admin (ÚNICO) en el cual se puede tomar medidas preventivas de seguridad en contra de usuarios con comportamientos inadecuados o/e inseguros.
